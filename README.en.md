# 基于SSM的学校运动会管理系统

#### Description
采用了Java语言和MySQL数据库来实现。系统按预定的算法完成了创办运动会、广播公告公示、赛事项目拟订、报名参赛、赛后成绩录入、查看比赛成绩、院系人员的信息存储、反馈建议、访问日记等功能。
